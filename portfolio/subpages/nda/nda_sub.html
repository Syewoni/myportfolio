Problem & Solution

ifame 태그 사용시 나타나는 오류

처음에는 html의 요소에 공유형 태그인 iframe을 이용하여 영상을 불러오고자 하였으나 영상 로드는 되었으나 콘솔창에서 보안문제로 인한 오류가 나타났습니다. YT player.js를 활용하여 유튜브 영상을 불러올 수 있었습니다.

가로로 움직이는 텍스트

처음에는 css의 animate를 활용하여 가로로 움직이는 텍스트를 구현하였으나, 텍스트가 끝나면 일정시간 후 다시 텍스트가 움직이는 것을 확인하였습니다. 바로 다음 텍스트가 나오는 것을 구현하려고 했기에, 다른 방법을 시도하다가 marquee 플러그인을 활용하여 끊기지 않고 움직이는 텍스트를 구현하였습니다.

서브메뉴가 클릭이 되지 않는 문제

PC버전 메뉴에서 :hover시 서브메뉴가 드롭다운 형태로 내려오게끔 구현하고자 하였으나 서브메뉴 클릭 시 서브메뉴 창이 닫기는 문제를 확인할 수 있었습니다. 서브메뉴에 margin값으로 인한 문제라는 것을 확인하고 서브메뉴에 margin이 아닌 padding값을 주어 해결하였습니다.

모바일, 태블릿 버전에서는 메인메뉴 클릭 시 해당하는 서브메뉴를 확인할 수 있도록 slideToggle( ) 을 사용하여 구현하였습니다.

Review

다채롭고, 화려한 웹 페이지를 만들어보고자 시도하였습니다. 원본 홈페이지와 다른 레이아웃을 선택했기 때문에 컬러 선정부터 스크립트 적용, 플러그인 적용까지 많은 시행착오를 경험해볼 수 있었습니다. 페스티벌이 주는 화려하고 다채로운 느낌을 살리기 위해 주요 컬러를 높은 채도의 컬러로 사용하며 디자인을 시도하였습니다.

기존 홈페이지에서는 이미지 슬라이드만 있어 메인페이지에 어떤 컨텐츠를 넣을까 고민하며 웹 페이지의 구조를 파악하고 주요 컨텐츠를 찾아보는 과정도 좋은 경험이 되었다고 생각합니다. 동작을 구현하기 위해 많은 공부를 하고, marquee, ytPlayer 등 플러그인을 적용시켜볼 수 있었습니다.
